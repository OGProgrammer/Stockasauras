Stockasauras adds short codes that allows you to tap into the powerful Yahoo Finance API. Get quotes, charts, news, and more for Stocks, Indices, and Currencies.

== Description ==
Stockasauras adds short codes to display real-time stock quotes, charts, and news into WordPress posts.

= Stock Quotes =

`[stockasauras_quote s=\"MSFT\"]`

Generates a stock quote `44.69`.

= Stock Charts =

`[stockasauras_chart symbol=\"MSFT\"]`

Generates a price chart.

== Installation ==

1. Upload `stockasauras.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Use the short codes in posts to show quotes, charts, or news.
